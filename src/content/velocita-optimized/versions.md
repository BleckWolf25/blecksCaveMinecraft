## **Understanding Velocita Edition Versions**

Velocita Edition’s versioning strategy ensures adaptability to diverse user needs and hardware configurations:

- **Standard Versions (e.g., Fabric 1.21.1):** Exclusively focused on optimization mods, these editions prioritize performance and efficiency.
- **++ Versions (e.g., ++Fabric 1.21.1):** Combines optimization with enhancements, such as improved user interfaces, visual tweaks, and quality-of-life features.

> [!NOTE]
> This distinction pertains exclusively to mods and does not extend to included resource packs or shaders, which remain uniform across all editions.

### Modrinth Versioning

Due to the development and upload sequence, older versions may appear first on Modrinth. Support for legacy versions was introduced after completing stable iterations for newer Minecraft versions. While unconventional, this approach ensures comprehensive backward compatibility.
